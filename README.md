## ⚙️ Running the Portfolio Locally

The portfolio is bundled as a clean, static single-page application. You can host and run it locally with any simple HTTP server:

1. Clone this repository:
   ```bash
   git clone https://github.com/Napa-Guna-Sai-Sujith/portfolio.git
   cd portfolio
   ```
2. Start a local server (e.g. using Python):
   ```bash
   python -m http.server 3000
   ```
3. Open **[http://localhost:3000](http://localhost:3000)** in your web browser.
